# Examples

This section contains process graphs that show how the openEO processes work in practice.

  • POC: Proof-of-concept use-cases, including API requests
  • NDVI-UC1: Deriving maximum NDVI measurements over pixel time series
  • ZONAL-UC3: Compute time series of zonal (regional) statistics over user-specified polygons
  • EVI: Deriving minimum EVI measurements over pixel time series

Please feel encouraged to add your examples via Pull Requests!
